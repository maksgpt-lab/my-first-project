<b>Кейс: регламент для нового сотрудника — за 15 минут</b>

Владелец небольшой студии дизайна нанял нового менеджера по работе с клиентами.

Обычно онбординг занимал неделю: объяснения, исправления ошибок, повторные вопросы. Стандартная история для малого бизнеса без HR.

На этот раз он попробовал по-другому.

<b>Что сделал:</b>
Описал процесс работы с клиентом своими словами — 7 шагов от первого контакта до сдачи проекта. Вставил в ChatGPT с запросом написать регламент для нового сотрудника.

<b>Результат через 15 минут:</b>
— Пошаговый регламент с ответственными и сроками
— Чек-лист для каждого этапа
— Раздел "типичные ошибки которых нужно избегать"

Менеджер вышел на работу, прочитал документ и задал 2 вопроса вместо обычных 20.

Руководитель сэкономил неделю объяснений. И получил документ который теперь работает для каждого нового найма.
