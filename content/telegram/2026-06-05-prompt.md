<b>Промпт дня: скрипт продаж под любую нишу</b>

<code>Ты опытный тренер по продажам. Напиши скрипт для обработки входящего запроса от потенциального клиента. Мой продукт/услуга: [описание]. Типичный клиент: [кто обращается, какая у него боль]. Цель звонка/диалога: [назначить встречу / закрыть сделку / отправить КП]. Включи: приветствие, 3 вопроса для выявления потребности, короткую презентацию ценности (2-3 предложения), ответ на возражение "я подумаю". Тон: [деловой / дружелюбный].</code>

<b>Как использовать:</b>
Получи скрипт → прочитай вслух → замени слова которые "не твои" → используй в следующем звонке.

Скрипт — не текст для чтения по бумажке. Это структура которую знаешь наизусть.
