<b>Промпт: скрипт продаж под твою нишу</b>

Скрипт продаж — это не текст для чтения по бумажке. Это структура которую знаешь наизусть и адаптируешь на ходу.

ChatGPT генерирует рабочий черновик за 2 минуты:

<code>Ты опытный тренер по продажам. Напиши скрипт для обработки входящего запроса. Мой продукт/услуга: [описание]. Типичный клиент: [кто обращается, какая боль]. Цель: [назначить встречу / закрыть сделку / отправить КП]. Включи: приветствие, 3 вопроса для выявления потребности, короткую презентацию ценности, ответ на возражение «я подумаю». Тон: [деловой / дружелюбный].</code>

Получи скрипт → прочитай вслух → замени слова которые «не твои» → используй в следующем звонке.
