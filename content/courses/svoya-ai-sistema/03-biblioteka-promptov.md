## Почему библиотека промптов — это актив

У меня в работе есть документ с промптами — я называю его "мой арсенал". Там собраны промпты под все задачи которые я делаю регулярно: для агентов, для анализа, для общения с клиентами, для технических заданий.

Я его собирал постепенно. Каждый раз когда нашёл промпт который даёт отличный результат — добавляю. Каждый раз когда что-то не работает — улучшаю.

Через 3 месяца такой работы у тебя 20-30 проверенных промптов под твой конкретный бизнес. Это не просто удобство — это актив. Новый сотрудник может взять этот документ и работать с AI на твоём уровне с первого дня.

---

## Структура библиотеки

Разбей промпты по категориям. Стандартные для малого бизнеса:

**Маркетинг и контент**
- Пост для Telegram / Instagram
- Контент-план на месяц
- Заголовки и описания для рекламы
- Email-рассылка

**Продажи**
- Скрипт входящего звонка
- Ответ на возражение "дорого"
- Персональное коммерческое предложение
- Follow-up после встречи

**Операционка**
- Должностная инструкция
- Регламент процесса
- ТЗ для сотрудника или подрядчика

**Коммуникация**
- Официальное письмо партнёру
- Ответ на жалобу клиента
- Отказ с сохранением отношений

## Формат хранения промпта

Каждый промпт в библиотеке должен содержать:

```
Название: [короткое, понятное название]
Когда использовать: [1-2 предложения]
Контекстный блок: [описание бизнеса, которое вставляется в начало]
Промпт: [сам текст с [placeholder] для переменных]
Примечания: [что менять под разные ситуации]
```

---

## Контекстный блок — основа всего

Самый важный элемент библиотеки — не промпты, а контекстный блок. Это описание твоего бизнеса, которое ты вставляешь в начало каждого диалога:

> Контекст моего бизнеса:
> Компания: [название]
> Сфера: [что делаете]
> Аудитория: [кто клиенты — возраст, должность, боли, желания]
> Тон коммуникации: [дружелюбный / деловой / экспертный]
> Продукт/услуга: [описание того что продаёте]
> Главное отличие от конкурентов: [что делает вас особенными]
> Чего мы никогда не делаем: [ограничения, которые важно соблюдать]

Напиши этот блок один раз. Вставляй перед любым промптом — и ChatGPT каждый раз работает с полным пониманием твоего бизнеса.

---

## Промпт для создания промпта

ChatGPT помогает писать хорошие промпты — это мета-использование, которое экономит часы:

> Я хочу создать шаблонный промпт для задачи: [описание задачи]. Я буду использовать его регулярно, подставляя разные [переменные: тему, клиента, продукт — нужное]. Напиши шаблон промпта с placeholder'ами в квадратных скобках. Промпт должен давать высококачественный результат без дополнительных инструкций.

## Пример библиотеки для юридической компании

**Промпт 1: Пост для Telegram**
```
[Контекстный блок]
Напиши пост для Telegram-канала о теме: [ТЕМА].
Структура: зацепляющее начало без "Привет!" → факт или история → практический совет → призыв к действию.
Объём — 180-220 слов. Тон — экспертный но без юридического жаргона.
```

**Промпт 2: Ответ на жалобу клиента**
```
[Контекстный блок]
Клиент написал жалобу: [ТЕКСТ ЖАЛОБЫ].
Напиши ответ: признай проблему, объясни что произошло (без самооправдания), предложи конкретное решение, сохрани отношения. Тон — профессиональный, без защитной позиции. 100-150 слов.
```

**Промпт 3: Вакансия**
```
[Контекстный блок]
Напиши текст вакансии на должность: [ДОЛЖНОСТЬ].
Включи: короткое описание компании (2-3 предложения), чем предстоит заниматься (6-8 пунктов), что важно для нас в кандидате (4-5 пунктов), что мы предлагаем (3-4 пункта), как откликнуться. Тон — честный, без корпоративного булшита.
```

## Попробуй прямо сейчас

1. Напиши свой контекстный блок — описание бизнеса по шаблону выше (10 минут)
2. Выбери 3 задачи, которые делаешь каждую неделю
3. Для каждой попроси ChatGPT создать шаблонный промпт
4. Сохрани всё в заметках (Notion, Apple Notes, Google Docs — неважно)

После этого урока у тебя есть основа библиотеки. В следующих уроках добавим автоматизацию и делегирование.
