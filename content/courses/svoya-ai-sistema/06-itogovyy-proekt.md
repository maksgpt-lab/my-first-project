## Это урок-практика, не урок-лекция

К этому моменту у тебя есть:
- Понимание что такое AI-система
- Аудит бизнеса с приоритетами
- Библиотека промптов (хотя бы начало)
- Автоматизированные процессы
- Понимание как делегировать AI команде

Финальный урок — собрать всё это в единую работающую систему. Не в теории, а прямо сейчас.

---

## Структура твоей AI-системы

Заполни этот документ — он и есть твоя система:

### Блок 1: Контекст бизнеса
```
Компания: ___
Сфера: ___
Аудитория: ___
Тон: ___
Главный продукт: ___
Отличие от конкурентов: ___
Что никогда не делаем: ___
```

### Блок 2: Топ-10 промптов (по приоритету)
Для каждого:
```
Название: ___
Когда использую: ___
Промпт: ___
```

### Блок 3: AI-процессы
```
Задача → Промпт → Ответственный → Периодичность → Критерий качества
```

### Блок 4: Что проверяю в результатах AI
```
Факты: ___
Тон: ___
Задача решена: ___
```

## Промпт для создания всей системы за один раз

Если хочешь сгенерировать черновик системы быстро:

> Ты мой бизнес-консультант по AI. Я прошёл курс и хочу запустить AI-систему для своего бизнеса. Мой бизнес: [описание]. Основные задачи где использую AI: [перечисли 5–7]. Помоги мне создать: 1) контекстный блок бизнеса, 2) список из 10 приоритетных промптов с шаблонами, 3) расписание AI-задач по дням недели, 4) чек-лист для контроля качества результатов. Оформи в виде документа который я могу сохранить и использовать ежедневно.

---

## Чек-лист запуска системы

Перед тем как считать систему запущенной:

- [ ] Контекстный блок написан и протестирован
- [ ] Топ-5 промптов работают и дают качественный результат
- [ ] Хотя бы 2 задачи переведены в регулярный AI-процесс
- [ ] Сотрудники (если есть) знают какие задачи делать с AI
- [ ] Есть место для хранения библиотеки (Notion / Google Docs)
- [ ] Запланировано время раз в месяц для обновления промптов

## Поздравляем с завершением программы

Ты прошёл путь от "что такое ChatGPT" до собственной AI-системы для бизнеса.

**Что у тебя теперь есть:**
- Понимание AI-инструментов и как они работают
- Навык написания промптов под любую задачу
- Рабочая система для маркетинга: контент, тексты, реклама
- Скрипты и инструменты для продаж
- Регламенты и процессы для операционки
- Личная AI-система, которая работает на тебя каждый день

**Что дальше:**
AI-инструменты развиваются быстро. Следи за обновлениями курса — мы добавляем новые уроки по мере появления важных инструментов и техник.

И помни: самая сложная часть уже позади — ты начал.
