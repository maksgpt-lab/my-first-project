# AI для бизнеса

Образовательная платформа на русском языке — учим автоматизировать бизнес-процессы с помощью AI без программирования.

## Что это

Практические курсы и гайды по AI-инструментам для предпринимателей, руководителей и команд. Никакого кода — только реальные инструменты и сценарии из бизнеса.

**Позиционирование:** "Научись автоматизировать бизнес-процессы с помощью AI — без программирования, на русском языке."

## Для кого

- Предприниматели и владельцы малого бизнеса
- Руководители и операционные менеджеры
- Маркетологи, HR, администраторы
- Все, кто хочет работать быстрее с помощью AI — без навыков разработки

## Основные функции

### Курсы и уроки
- Структурированные курсы с делением на модули и уроки
- Контент в формате MDX — удобно писать и обновлять
- Прогресс обучения для каждого пользователя

### Темы курсов
- ChatGPT и Claude для ежедневной работы
- Автоматизация с n8n и Make (без кода)
- AI-ассистенты для команды
- Готовые промпты под задачи бизнеса

### Навигация по контенту
- Каталог всех курсов
- Структура: `/courses` → `/courses/[курс]` → `/courses/[курс]/[урок]`

## Монетизация

### Модель: Freemium + Telegram-клуб

**Бесплатно** (трафик и доверие):
- Вводные уроки каждого курса на сайте
- Telegram-канал: советы, промпты, новости AI

**Платно** (монетизация):
- Полный доступ к курсам — разовая оплата за курс (2,000–4,000 ₽)
- Telegram-клуб — закрытое сообщество (500–1,000 ₽/мес или 5,000 ₽/год)
  - Q&A с автором
  - Готовые шаблоны и промпты
  - Разборы кейсов подписчиков
  - Ранний доступ к новым курсам

### Воронка
> Telegram-канал → сайт с бесплатными уроками → платный курс или Telegram-клуб

## Маркетинг и удержание (по Котлеру)

### Лестница лояльности

**Незнакомец → Посетитель**
- SEO-статьи: "как автоматизировать X с помощью AI"
- Бесплатные лид-магниты: чек-листы промптов, шаблоны по ролям (маркетолог, HR, руководитель)
- Telegram-канал как главная точка входа
- Коллаборации с бизнес-сообществами и подкастами

**Посетитель → Лид**
- Email-подписка в обмен на полезный материал (не просто "подпишись на новости")
- Подписка на Telegram-канал = лид
- Бесплатный вводный мини-курс с регистрацией

**Лид → Покупатель**
- Кейсы и истории успеха реальных студентов
- Первый модуль каждого курса бесплатно
- Ограниченные офферы для новых подписчиков

**Покупатель → Постоянный**
- Новые курсы и регулярные обновления контента
- Персонализация по роли: разные треки для маркетолога, HR, руководителя
- Telegram-клуб как место, где хочется оставаться

**Постоянный → Евангелист**
- Реферальная программа: привёл друга — получи месяц клуба бесплатно
- Сертификаты за курсы — люди делятся в соцсетях
- Истории успеха студентов на сайте

### Решение проблемы устаревания контента
- Дата последнего обновления на каждом уроке
- Кнопка "Нашёл устаревшее" — читатели сигнализируют об изменениях
- Квартальный аудит всех материалов
- Changelog — страница с историей обновлений (даёт повод вернуться)

### Удержание пользователей (технически)
- Прогресс через localStorage — без регистрации, "продолжить с места"
- Streak (серия дней) — мотивирует заходить регулярно
- Email-рассылка при выходе новых уроков
- Telegram-канал — главный инструмент возврата аудитории

## Конкурентное преимущество

- **Русский язык** — минимальная конкуренция в нише AI для бизнеса
- **Без кода** — фокус на инструментах, доступных любому
- **Практика** — реальные сценарии из бизнеса, а не теория

## Технический стек

- **Frontend:** Next.js + Tailwind CSS
- **Контент:** MDX (Markdown + JSX компоненты)
- **Деплой:** Vercel
